01. Sobre Mim

Sou desenvolvedor backend apaixonado por criar sistemas robustos e escaláveis. Minha expertise está em Python e frameworks modernos como FastAPI, Django e Flask, desenvolvendo APIs RESTful que impulsionam aplicações web.

Também trabalho com desenvolvimento frontend usando HTML, CSS e JavaScript, criando interfaces funcionais e responsivas. Tenho experiência com Node.js, onde desenvolvi bots de WhatsApp utilizando a biblioteca Baileys, automatizando processos e integrando sistemas de mensageria.

Com experiência em projetos acadêmicos e pessoais, já desenvolvi desde sistemas de autenticação completos até APIs de alta performance e automações inteligentes. Domino ferramentas essenciais como Git, GitHub e Linux, aplicando boas práticas de desenvolvimento e arquitetura de software.

Atualmente estudo Informática com foco em desenvolvimento backend, aprimorando constantemente minhas habilidades em design de APIs, otimização de bancos de dados e práticas DevOps. Busco sempre escrever código limpo, testável e bem documentado.
